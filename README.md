# ⚡ Inverter Digital Twin

Real-time digital twin of a grid-connected single-phase VSI 
with LC filter, SPWM modulation, PLL sync and fault detection.

## Features
- State-space inverter model (L, C, R parameters)
- Live waveforms: Vout, Inductor Current, SPWM output
- Fault injection: Inductor, Capacitor, High ESR
- Residual-based fault detection engine
- PLL grid synchronisation

## Tech Stack
- Python (numpy, scipy)
- React + Recharts (live dashboard)
